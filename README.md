# HK Higher Diploma Graduates — Overseas Top-up Degree Information System

An MS Access database application designed 

## What It Does
Helps Hong Kong Higher Diploma graduates find overseas top-up degree programmes based on GPA, budget, visa policy, living cost, and career prospects.

## Features
- 6 related tables (Country, University, Programme, Student, Application, CareerProspect)
- 2 input masks (Phone Number, Date)
- 2 validation rules (GPA range, Tuition > 0)
- 2 data entry forms
- 1 summary report
- 10 SQL queries (SELECT, Sorting, Like, Number/Date/Boolean conditions, AND/OR, Group By, Join, Calculated Field)
- Star rating enhancement for user-friendly score display

## Tools Used
- Microsoft Access
- draw.io (for ERD)
